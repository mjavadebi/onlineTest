Researcher Setup Guide

This guide helps you create your own customized test platform and save responses to your Google Sheet.

Steps to Get Started

  1. Clone or fork this repository from GitHub:
    https://github.com/mjavadebi/onlineTest
  2. Modify the test HTML (e.g., felder.html):
    Replace the fetch URL with your own Google Apps Script Web App URL to save data.
  3. Create a Google Sheet to store responses:
    Set up columns for the dimensions (e.g., ACT_REF, SEN_INT, VIS_VRB, SEQ_GLO) and timestamps.
  4. Create a Google Apps Script Web App:
    Use this sample script and publish it as a Web App accessible by anyone: 
    function doPost(e) {
  var sheet = SpreadsheetApp.openById("YOUR_SHEET_ID").getSheetByName("Sheet1");
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([new Date(), data.ACT_REF, data.SEN_INT, data.VIS_VRB, data.SEQ_GLO]);
  return ContentService.createTextOutput("Success");
}
  5. Deploy your GitHub repository using GitHub Pages
    Your tests will be live and will send data to your Google Sheet.
